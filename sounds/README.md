# Sound Effects

Place audio files here:

- correct.mp3  — played when a letter lands in the right slot
- wrong.mp3    — played when a letter is dropped in the wrong slot
- levelup.mp3  — played when a level is completed

## Notes
- Keep files short (under 1 second for correct/wrong, 2-3 seconds for levelup)
- Free sources: freesound.org, pixabay.com/sound-effects, opengameart.org
- The game works without these files (Web Speech API handles voice feedback)
